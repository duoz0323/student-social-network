# Workflow sửa lỗi

## Bước 1: Phân tích nguyên nhân cốt lõi

Không đưa code sửa ngay.

Phải trình bày:

- Vị trí lỗi.
- Nguyên nhân trực tiếp.
- Nguyên nhân gốc.
- Điều kiện tái hiện.
- Phạm vi ảnh hưởng.

## Bước 2: Đề xuất giải pháp

Trình bày:

- Giải pháp ưu tiên.
- Lý do.
- File cần sửa.
- Rủi ro.
- Cách kiểm thử.

## Bước 3: Sửa code

- Sửa nguyên nhân gốc.
- Code hoàn chỉnh.
- Comment tiếng Việt.
- Không tạo regression.
- Không thay đổi kiến trúc không cần thiết.

## Bước 4: Kiểm thử lại

- Trường hợp lỗi ban đầu.
- Trường hợp hợp lệ.
- Trường hợp biên.
- Phân quyền.
- Dữ liệu rỗng.
